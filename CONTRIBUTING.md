# Contributing

This repository is a binary SDK and binding-integration repository. It does not
accept changes to the private `libdense_sim` or DenseDB implementation source.

Useful public contributions include:

- binding fixes that preserve the published C ABI;
- documentation corrections;
- packaging fixes;
- additional binding tests; and
- reproducible consumer-side integration reports.

Do not submit reverse-engineered or reconstructed core implementation source.
By contributing, you confirm that you have the right to provide the submitted
material and that it may be distributed under the repository license.
